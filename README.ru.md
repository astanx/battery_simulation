## Как использовать

### Установка

Чтобы запустить симуляцию, вам нужно либо клонировать репозиторий, либо скачать код.

```bash
git clone https://github.com/astanx/battery_simulation
cd battery_simulation
```

Перед установкой зависимостей вы можете создать виртуальное окружение для Python

##### Windows
```bash
python -m venv venv
venv\Scripts\activate
```

##### MacOS/Linux
```bash
python -m venv venv
source venv/bin/activate
```

#### Установка зависимостей

```bash
pip install -r requirements.txt
```

Перед запуском кода вы можете изменить настройки в файле [config.py](./config.py), например:

```python
useUI = True
```

Чтобы задать конкретные значения температуры и влажности для квадрата  
Чтобы запустить симуляцию, выполните:

```bash
python main.py
```

---

### Управление

Вы можете управлять симуляцией с помощью клавиатуры

- ESCAPE: закрыть симуляцию
- SPACE: остановить симуляцию, но заряд всё равно будет расходоваться
- ENTER: приостановить симуляцию, всё будет на паузе
- ARROW_UP: переместить выбранную батарею на один квадрат вверх
- ARROW_DOWN: переместить выбранную батарею на один квадрат вниз
- ARROW_LEFT: выбрать батарею слева от текущей
- ARROW_RIGHT: выбрать батарею справа от текущей

### Конфигурация

Вы можете настраивать энергопотребление и поведение элементов, изменяя файлы:

- [src/cell/cell_constants.py](./src/cell/cell_constants.py)
- [src/cell/hydrogen_cell_constants.py](./src/cell/hydrogen_cell_constants.py)
- [src/cell/lithium_cell_constants.py](./src/cell/lithium_cell_constants.py)

Вы можете настраивать батареи и дорожки, изменяя [src/app.py](./src/app.py)

Также можно настроить приложение, изменяя [config.py](./config.py)

## Формулы

### Литиевая батарея

- #### [Зависимость типа Аррениуса от температуры](https://en.wikipedia.org/wiki/Arrhenius_equation)
  ##### Используется в коде
  ```python
  RT = R_ref * exp(E_a / R * (1/temperature - 1/T_ref))
  ```
  Модифицированная форма уравнения Аррениуса, используется для моделирования температурной зависимости внутреннего сопротивления и поведения реакций внутри батареи.

- #### [Напряжение холостого хода (OCV)](https://en.wikipedia.org/wiki/Open-circuit_voltage)
  ##### Используется в коде
  ```python
  OCV = OCV_ref + alpha * (self.SOC - 0.5) + g * (temperature - T_ref)
  ```
  OCV — это напряжение батареи при отсутствии тока. В коде оно аппроксимируется с помощью опорного OCV с поправками на SOC и температуру.

- #### [Эффективное внутреннее сопротивление](https://en.wikipedia.org/wiki/Internal_resistance)
  ##### Используется в коде
  ```python
  R = self.arrhenius_equation(temperature)
  ```
  Зависит от множества факторов, включая температуру и химию батареи. В коде аппроксимируется с помощью зависимости типа Аррениуса.

- #### [Поляризационное напряжение (перенапряжение)](https://en.wikipedia.org/wiki/Polarization_(electrochemistry))
  ##### Используется в коде
  ```python
  R_pol = R_ref * exp(E_pol / R * (1/temperature - 1/T_ref))
  V_pol = self.current * R_pol
  ```
  Поляризационное напряжение — дополнительные потери напряжения сверх омического сопротивления, возникающие при протекании тока из-за кинетических и диффузионных ограничений в электродах.

- #### [SOC (State of Charge) — степень заряженности](https://en.wikipedia.org/wiki/State_of_charge)
  ##### Используется в коде
  ```python
  d_SOC = (self.current * efficiency * dt) / (self.capacity * 3600)
  ```
  SOC показывает, какая доля заряда осталась относительно полной ёмкости.  
  В симуляции SOC уменьшается пропорционально протекающему току во времени.

### Водородная батарея

- #### [Уравнение Нернста](https://en.wikipedia.org/wiki/Nernst_equation)
  ##### Используется в коде
  ```python
  argument = (H * vapor) / (P_H2 * pow(P_O2, 0.5))
  E_nernst = E - (R * temperature / (2 * F)) * log(argument)
  ```
  В водородном топливном элементе показывает теоретическое максимальное напряжение при данных условиях.

- #### [Плотность обменного тока (типа Аррениуса)](https://en.wikipedia.org/wiki/Arrhenius_equation)
  ##### Используется в коде
  ```python
  I_O = I_O_ref * exp((-E_a / R) * (1/temperature - 1/T_ref))
  ```
  Выражение моделирует температурную зависимость плотности обменного тока по типу Аррениуса.

- #### [Уравнение Батлера–Фольмера](https://en.wikipedia.org/wiki/Butler%E2%80%93Volmer_equation)
  ##### Используется в коде
  ```python
  E = (R * temperature / (a * n * F)) * log(self.current_density / I_O)
  ```
  Упрощённая форма, отражающая отклонение плотности тока от обменной плотности из-за кинетики реакции.

- #### [Модель проводимости мембраны Springer](https://www.mdpi.com/2077-0375/10/11/310)
  ##### Используется в коде
  ```python
  sigma_mem = (0.005139 * l - 0.00326) * exp(1268 * (1/303 - 1/temperature)) * 100
  n_ohmic = self.current_density * (d_mem / sigma_mem)
  ```
      Описывает омические потери в протон-обменной мембране из-за сопротивления транспорту протонов.

- #### [Нернстовское концентрационное перенапряжение](https://www.mdpi.com/2076-3417/9/6/1066)
  ##### Используется в коде
  ```python
  I_lim = n * F * D_O2 * C_bulk / d_GDL
  C_surface = C_bulk * (1 - self.current_density / I_lim)
  E = (R * temperature / (n * F)) * log(C_bulk / C_surface)
  ```
  Моделирует концентрационные потери: при увеличении тока реагенты (например, кислород) расходуются у поверхности реакции быстрее, чем успевают диффундировать из объёма.

### Ощущаемая температура
 [Heat index](https://en.wikipedia.org/wiki/Heat_index) был использован для вычисления ощущаемой температуры
 ##### Используется в коде
 ```python
  HI = (-42.379 +
          2.04901523 * T +
          10.14333127 * H -
          0.22475541 * T * H -
          0.00683783 * T * T -
          0.05481717 * H * H +
          0.00122874 * T * T * H +
          0.00085282 * T * H * H -
          0.00000199 * T * T * H * H)
    
 if H < 13 and 80 <= T <= 112:
   adjustment = ((13-H)/4)*sqrt((17-abs(T-95))/17)
   HI -= adjustment
    
 if H > 85 and 80 <= T <= 87:
   adjustment = ((H-85)/10) * ((87-T)/5)
   HI += adjustment
 ```
